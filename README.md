# Employee_Payroll_System

## Project Overview

This system manages employee payroll for small and medium businesses. It handles employee records, attendance, salary calculations, and payroll reports.

## Features

* Manage employee details
* Track attendance
* Calculate salaries with bonuses and tax deductions
* Generate payroll records per month and year
* Produce payslips

## Technologies Used

* C#
* VS Code
* Visual Studio
* Git and GitHub

## Modules

* Maintain employee
* Maintain attence record
* Maintain deduction and bonus
* Payroll Processing
* Record employee attendance for payroll calculation
* Generate reports, including total salary per employee per time period

## How to Run

1. Clone repository
2. Open in VS Code
3. Build and run project
